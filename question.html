Question-2: What is the difference between innerHTML, innerText, and textContent ?

Differences Between innerHTML, innerText, and textContent is Use innerHTML → When inserting or modifying HTML content (but be cautious of XSS). Use innerText → When getting only the visible text of an element. Use textContent → When getting the raw text content, including hidden text, in the most efficient way.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach a single event listener to a parent and detect events based on the target element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts at the target element and propagates up through its ancestor elements until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Use appendChild() / prepend() for better performance. Use removeChild() / remove() when deleting elements. Avoid innerHTML for dynamic content (security risks).
